A janela Booking Manager apresenta, em tempo real, todas as reservas do Hotel. Aqui é possivel pesquisar, criar, editar ou executar ações sobre reservas já existente

![Booking Manager 01 Pn](/images/booking_Manager_01.png)

### Área de Filtros

Disponivel uma área de filtros onde é possível filtrar reservas de forma rápida e avançada. Esta ãrea pode ser expandida para revelar campos adicionais, permitindo uma pesquisa mais precisa de reservas com base em diversos critérios.

![image.png](/images/image.png)

<Note>
  Após definir os critérios de pesquisa, clique no botão "Search", canto superior direito,  para executar a pesquisa
</Note>
